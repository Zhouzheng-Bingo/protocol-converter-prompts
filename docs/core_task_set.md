# Core Task Set

The `prompts/core_directions/` folder covers the 12 representative directions used as the focused core task set:

1. HTTP -> Modbus TCP
2. HTTP -> OPC UA
3. Modbus TCP -> HTTP
4. MQTT -> Modbus TCP
5. MQTT -> OPC UA
6. OPC UA -> HTTP
7. HTTP -> CoAP
8. CoAP -> MQTT
9. MQTT -> CoAP
10. HTTP -> LoRaWAN
11. LoRaWAN -> MQTT
12. LoRaWAN -> OPC UA
